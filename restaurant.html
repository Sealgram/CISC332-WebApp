Restaurant Database Manager

Welcome to the restaurant database manager! In this web app you will find the functionality to:

  • List all the orders a customer made on a specific date
  • Add a new customer to the database
  • View the dates on which orders were placed, along with the number of orders on that date
  • View an employee's schedule from Monday to Friday
